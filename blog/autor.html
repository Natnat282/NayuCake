Como le comenté en clase, utilicé la herramienta Nicepage como base para crear mi proyecto. Para la página de inicio (Home), partí de una plantilla existente dentro de la aplicación, cuyo código de referencia es 6248996. Sin embargo, recalco que dicha plantilla fue completamente modificada para adaptarse a mis necesidades específicas.

Quiero destacar que las demás páginas del proyecto fueron diseñadas íntegramente por mí, tanto en términos de diseño como de estructura. Además, realicé modificaciones directamente desde Visual Studio para ajustar y personalizar aspectos específicos que necesitaba implementar.

También quiero enfatizar que estoy utilizando una API REST para el envío de correos y la conversión de dólares en la página de productos. Para la implementación, utilizo jQuery para el traductor, y las API del la galeria de productos se emplean de forma local.
